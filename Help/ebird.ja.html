eBird 記録 FAQ
eBird とは何ですか？ eBird は、コーネル大学鳥類学研究所が管理する、鳥の観察データを共有するためのオンラインプラットフォームです。
eBird 記録ファイルとは何ですか？ 記録ファイルは、eBird プラットフォームにインポートできる鳥の観察データのスプレッドシートです。
Chirpity はどのようにして私の検出を整理しますか？

アプリケーションは、エクスポートに関連付けられた各ファイルのチェックリストを作成します。これらのチェックリストには、そのファイル内の各種のコール検出の合計が含まれています。 コール数を入力した場合はその数値が使用され、そうでない場合は1コールと見なされます。

エクスポートされたファイルには、各列が何を表しているかを説明するヘッダーの行が含まれています。 詳細は eBird サイト で確認できます。

重要： ファイルをアップロードする前にヘッダー行を削除する必要があります。
eBird 記録ファイルをどのようにアップロードしますか？ eBird にアクセスして、eBird アカウントにサインアップまたはサインインします。アカウントを作成し、ログインしたら インポートフォーム に移動します。
  1. "ファイルを選択" をクリックして、準備したファイルを選択します
  2. 重要： "eBird 記録形式 (拡張)" をチェックします
  3. "ファイルをインポート" をクリックします
記録ファイルをアップロードしようとするとエラーが発生するのはなぜですか？

まず、ヘッダー行を削除したことを確認してください。eBird はファイルの最初の行に観察データが含まれていることを期待しています。

これが問題でない場合、種が認識されないというエラーが表示される場合は、Chirpity で設定した言語設定が eBird アカウントの設定と一致していることを確認してください。
国および州/県の列が空白なのはなぜですか？

これらのフィールドは eBird 記録形式ではオプションです。Chirpity が使用する OpenStreetMap の場所検索 API は州と国を提供しますが、eBird で使用される形式では提供しません。あなたの国と地域の正しい形式はこの スプレッドシート で確認できます。
なぜ eBird を選んだのですか？他のサイトと共有する計画はありますか？

eBird は最も簡単に使用できるプラットフォームです。これは、公開された記録形式があり、おそらく最も重要なことに、eBird の分類が BirdNET と Nocmig モデルの両方で使用される分類と一致しているためです。

将来のリリースで Xeno-Canto とオーディオクリップを共有できるようにすることを希望しています。これは、適切なアップロード API をリリースするかどうかに依存します。
記録ファイルの正確性を確認する責任はあなたにあります。Chirpity は記録ファイル内のデータの正確性について一切の保証を行いません。この機能を使用することにより、あなたはこれを認識し、同意するものとします。
