Часто задаваемые вопросы о записи eBird
Что такое eBird? eBird — это онлайн-платформа для обмена данными о наблюдениях за птицами, управляемая Лабораторией орнитологии Корнелла.
Что такое файл записи eBird? Файл записи — это электронная таблица с данными о наблюдениях за птицами, которую можно импортировать в платформу eBird.
Как Chirpity организует мои данные о детекции?

Приложение создает контрольный список для каждого файла, связанного с вашим экспортом. Эти списки содержат общее количество детекций вызовов для каждого вида в этом файле. Если вы указали количество вызовов, используется это число; в противном случае предполагается один вызов.

Экспортируемый файл содержит строку заголовков, объясняющую, что представляет каждая колонка. Вы можете найти дополнительные сведения на сайте eBird.

Важно: перед загрузкой файла необходимо удалить строку заголовка.
Как загрузить файл записи eBird? Перейдите на сайт eBird и зарегистрируйтесь или войдите в свою учетную запись. После входа в систему откройте форму импорта.
  1. Нажмите "Выбрать файл" и выберите подготовленный файл.
  2. Важно: отметьте "Формат записи eBird (расширенный)".
  3. Нажмите "Импортировать файл".
Почему появляются ошибки при попытке загрузить файл записи?

Сначала убедитесь, что вы удалили строку заголовка. eBird ожидает, что первая строка файла будет содержать наблюдение.

Если это не так, и вы видите ошибки, связанные с нераспознанными видами, убедитесь, что настройки языка в Chirpity соответствуют настройкам языка вашей учетной записи eBird.
Почему столбцы "Страна" и "Штат/Провинция" пусты?

Эти поля являются необязательными в формате записи eBird. Хотя API поиска местоположений OpenStreetMap, используемый Chirpity, предоставляет данные о штате и стране, он не формирует их в формате, используемом eBird. Вы можете найти правильный формат для вашей страны и региона в этой электронной таблице.
Почему вы выбрали eBird? Планируете ли вы интеграцию с другими платформами?

eBird — это самая простая в использовании платформа. Это связано с тем, что у нее есть опубликованный формат записи, и, возможно, самое важное, таксономия eBird соответствует таксономии, используемой моделями BirdNET и Nocmig.

Я надеюсь предложить возможность делиться аудиоклипами с Xeno-Canto в будущем обновлении. Это зависит от их выпуска подходящего API для загрузки.
Ответственность за проверку точности файла записи лежит на вас. Chirpity не гарантирует точности данных в файлах записи. Используя эту функцию, вы подтверждаете и соглашаетесь с этим.
