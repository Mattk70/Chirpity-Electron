eBird-записи часто задаваемые вопросы
Что такое eBird? eBird это онлайн-платформа для обмена данными наблюдений за птицами, управляемая Корнеллской лабораторией орнитологии 'Cornell Lab of Ornithology'.
Что такое файл записей eBird? Файл записи представляет собой электронную таблицу данных наблюдений за птицами, которые могут быть импортированы в платформу eBird.
Как Chirpity организует мои обнаружения

Приложение создает контрольный список для каждого аудиофайла, связанного с вашим экспортом данных. Эти контрольные списки содержат общее количество обнаруженных объектов для каждого вида в этом файле. Если вы ввели количество вокализаций, используется эта цифра, в противном случае предполагается один вызов.

Экспортированный файл содержит строку заголовков, объясняющих, что представляет собой каждый столбец. Вы найдете более подробная информация на сайте eBird.

Важно: строка заголовка должна быть удалена перед загрузкой файла.
Как мне загрузить свой файл с записью eBird? Посетить eBird и зарегистрируйтесь в своей учетной записи eBird. Как только вы создадите учетную запись и войдете в систему, перейдите на форма импорта.
  1. Нажмите "Выбрать файл" ("Choose file") и выберите файл, который вы подготовили
  2. Важно: установите флажок "Формат записи eBird (расширенный)" ("eBird Record format (Extended)")
  3. Нажмите "Импортировать файл" ("Import file")
Почему при попытке загрузить свой файл записи возникают ошибки?

Во-первых, убедитесь, что вы удалили строку заголовка. eBird ожидает, что первая строка в файле будет содержать наблюдение.

Если проблема не в этом, а в том, что вы видите ошибки, связанные с невозможностью распознавания видов, убедитесь, что языковые настройки, которые вы установили в Chirpity, соответствуют языковым настройкам, установленным в вашей учетной записи eBird.
Почему столбцы 'Страна' и 'Штат/провинция' не заполнены?

В формате записей eBird эти поля необязательны. Хотя API поиска местоположения на Openstreetmap, используемый Chirpity, предоставляет данные о штате и стране, он не предоставляет их в формате, используемом eBird. Почему вы выбрали eBird? Планируете ли вы разрешить совместное использование с другими сайтами?

eBird - самая простая в использовании платформа. Это связано с тем, что она имеет формат опубликованных записей, и, возможно, самое главное, таксономия eBird совпадает с таксономией, используемой в моделях BirdNET и Nocmig.

Я надеюсь предложить возможность делиться аудиоклипами с Xeno-Canto в следующем выпуске. Это зависит от того, выпустят ли они подходящий API для загрузки.
Ответственность за проверку файла записей на точность лежит на вас. Chirpity не дает никаких гарантий относительно точности каких-либо данных, содержащихся в файлах записей. Используя эту функцию, вы подтверждаете и соглашаетесь с этим.
