eBird记录常见问题解答
什么是 eBird？ eBird 是一个在线平台，用于共享鸟类观察数据，由康奈尔鸟类学实验室管理。
什么是 eBird 记录文件？ 记录文件是一个包含鸟类观察数据的电子表格，可导入 eBird 平台。
Chirpity 如何组织我的检测结果？

该应用程序为与您的导出相关的每个文件创建一份清单。这些清单包含该文件中每个物种呼叫检测的总数。 如果您输入了呼叫数量，则使用该数字，否则假设为一次呼叫。

导出的文件包含一行标题，解释每列的含义。 您可以在 eBird 网站上找到更多详细信息。

重要提示：上传文件前需要删除标题行。
如何上传我的 eBird 记录文件？ 访问 eBird，注册或登录您的 eBird 帐户。登录后，前往 导入表单。
  1. 点击“选择文件”，然后选择您准备好的文件。
  2. 重要提示：勾选“eBird Record format (Extended)”。
  3. 点击“导入文件”。
为什么上传记录文件时会出现错误？

首先，请确保您已经删除了标题行。eBird 期望文件的第一行包含观察记录。

如果这不是问题，并且您看到与物种无法识别相关的错误，请确保您在 Chirpity 中设置的语言偏好与 eBird 帐户中设置的语言相匹配。
为什么国家和州/省列是空的？

这些字段在 eBird 记录格式中是可选的。尽管 Chirpity 使用的 OpenStreetMap 地点查询 API 可以提供州和国家的信息，但它未按 eBird 使用的格式提供这些信息。 您可以在此电子表格中找到您所在国家和地区的正确格式。
为什么选择 eBird？是否计划支持其他网站的共享？

eBird 是最简单的平台，因为它有一个已发布的记录格式，也因为 eBird 的分类法与 BirdNET 和 Nocmig 模型使用的分类法一致。

我希望在未来的版本中提供与 Xeno-Canto 共享音频剪辑的功能。这取决于他们是否发布合适的上传 API。
检查记录文件准确性的责任在于您自己。Chirpity 对记录文件中的数据准确性不作任何保证。 使用此功能即表示您承认并同意这一点。
