File handling
Ctrl+O Show file open dialog
Ctrl-S Save detections to the records archive.
Analysis
Ctrl-A Analyse the current file.
Shift-Ctrl-A Analyse all open files.
Esc Stop analysis
Ctrl-Tab Toggle between exploring the archive and the latest analysis results. This allows you to compare current detections with those you've previously recorded.
Transport Controls
SpaceBar Toggle Play/Pause the audio
P Play the selected region. Only active if a region has been selected.
Ctrl-C Centre the spectrogram at the current playhead position.
Ctrl-G Go to position in file
Home (Fn + ArrowLeft) Move the playhead to the beginning of the file
End (Fn + ArrowLeft) Move the playhead to the end of the file
PgUp (Fn + ArrowUp) Move the playhead back by a screen width
PgDn (Fn + ArrowDown) Move the playhead forward by a screen width
Left arrow (←) Move the playhead forward.
Right arrow (→) Move the playhead backward.
Tab or Down Arrow (↓) Skip to the next detection
Shift Tab or Up Arrow (↑) Skip back to the previous detection
Spectrogram
Plus (+) Zoom in on the spectrogram. When zooming, Chirpity automatically adjusts the short-time Fourier transform (STFT) window size to optimize the spectral resolution
Minus (-) Zoom out the spectrogram. When zooming, Chirpity automatically adjusts the STFT window size to optimize the spectral resolution
Ctrl Plus (+) Manually reduce the spectrogram window size. Adjusting the window size has the effect of sharpening / blurring the calls in the spectrogram.
Ctrl Minus (-) Manually increase the spectrogram window size
Ctrl-T Toggle between Timecode and Time of Day
Records
Delete or Backspace Delete current record
Ctrl-Z Restore deleted record(s)
Ctrl-V Validate the current active record (confirm the detection)
