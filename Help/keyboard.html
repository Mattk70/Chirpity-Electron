File handling
Ctrl-O Show file open dialog
Ctrl-S Save detections as a CSV file.
Ctrl-A Save label file for Audacity. Only active after file analysis, when there are labels to save!
Esc Abort an operation. Effective for file loading and analysis
Transport Controls
SpaceBar Toggle Play/Pause the audio
P Play the selected region. Only active if a region has been selected.
Home Move the playhead to the beginning of the file
End Move the playhead to the end of the file
PgUp Move the playhead back by a screen width
PgDn Move the playhead forward by a screen width
Left arrow (>) Move the playhead forward. Can be used to shift the screen view if a call is cut off at the screen edges
Right arrow (<) Move the playhead backward. Can also be used to shift the screen view if a call is cut off at the screen edges
Ctrl+ or + Zoom in on the spectrogram. When zooming, Chirpity automatically adjusts the STFT window size to optimize the spectral resolution
Ctrl- or - Zoom out the spectrogram. When zooming, Chirpity automatically adjusts the STFT window size to optimize the spectral resolution
Tab Skip to the next detection
Shift Tab Skip back to the previous detection
