FAQ do Registro eBird
O que é o eBird? eBird é uma plataforma online para compartilhar dados de observação de aves, gerenciada pelo Cornell Lab of Ornithology.
O que é um Arquivo de Registro eBird? O arquivo de registro é uma planilha com dados de observação de aves que pode ser importada para a plataforma eBird.
Como o Chirpity organiza minhas detecções?

O aplicativo cria uma lista de verificação para cada arquivo associado à sua exportação. Essas listas contêm o total de detecções de chamadas para cada espécie nesse arquivo. Quando você insere uma contagem de chamadas, esse número é usado; caso contrário, presume-se uma chamada.

O arquivo exportado contém uma linha de cabeçalhos explicando o que cada coluna representa. Você pode encontrar mais detalhes no site do eBird.

Importante: a linha de cabeçalho precisa ser excluída antes de carregar o arquivo.
Como faço para carregar meu arquivo de registro eBird? Acesse eBird e crie uma conta ou faça login. Depois de entrar, vá para o formulário de importação.
  1. Clique em "Escolher arquivo" e selecione o arquivo que você preparou.
  2. Importante: marque "Formato de Registro eBird (Estendido)".
  3. Clique em "Importar arquivo".
Por que recebo erros ao tentar carregar meu arquivo de registro?

Primeiro, certifique-se de que você excluiu a linha de cabeçalho. O eBird espera que a primeira linha do arquivo contenha uma observação.

Se esse não for o problema e você estiver vendo erros relacionados a espécies não reconhecidas, certifique-se de que as preferências de idioma definidas no Chirpity correspondam às do seu perfil no eBird.
Por que as colunas País e Estado/Província estão em branco?

Esses campos são opcionais no formato de registro eBird. Embora a API de pesquisa de locais do OpenStreetMap usada pelo Chirpity forneça informações sobre Estado e País, ela não os apresenta no formato usado pelo eBird. Você pode encontrar o formato correto para seu país e região nesta planilha.
Por que você escolheu o eBird? Há planos para compartilhar com outros sites?

O eBird é a plataforma mais simples de usar. Isso porque ele tem um formato de registro publicado e, talvez mais importante, a taxonomia do eBird é compatível com a utilizada pelos modelos BirdNET e Nocmig.

Espero oferecer a possibilidade de compartilhar clipes de áudio com o Xeno-Canto em uma futura atualização. Isso depende de eles lançarem uma API de upload adequada.
A responsabilidade de verificar a precisão do arquivo de registro é sua. O Chirpity não oferece garantias para a exatidão de nenhum dado nos arquivos de registro. Ao usar este recurso, você reconhece e concorda com isso.
